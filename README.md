# pokethnos

Fan-made do jogo de tabuleiro Ethnos com temática Pokémon.

## Como jogar

1. Clone o repositório:
   ```bash
   git clone https://github.com/danilloclopes/pokethnos.git
   ```
2. Abra o arquivo `pokethnos.html` no navegador para iniciar o jogo.
3. Consulte o arquivo `regras.html` para ler as regras completas.
